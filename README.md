# music-player
Grounded YouTube Search: Powered by the Gemini API with Google Search grounding. Enter any song name, and Gemini searches the live web to fetch real, active YouTube video IDs.

Vibe-Based AI Playlists: Describe a mood (e.g., "chill lo-fi rain," "80s synthwave workout"), and Gemini will curate and search a custom 5-song playlist for you.

AI Lyrics Generator: Generates and formats lyrics on the fly for the currently playing track.

YouTube Player API Integration: A completely customized playback control deck (Play/Pause, Seek bar, Volume slider, Mute, Shuffle, Repeat, Previous/Next, and Queue Management).

Video Mode: Toggle a elegant float
